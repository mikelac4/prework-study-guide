HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items.
  • The meta and title elements are nested inside of the head element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declarations contain the CSS property you want to apply and the value of the property.
  • In the img selector the displauy property is assigned to the block value which assigns each image to start on a new line.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: the add command adds modifications in the current working branch while the -A flag indicates to add all changes.
  • git commit -m "example": describes what the commit contains.
  • git pull origin main: checks to see if the local branch is in sync with the base branch in GitHub.
  • git push origin "branch name": pushes local changes to the remote branch in GitHub.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements and loops can be used to control the order in which code is excuted.
  • An array is a single variable used to hold a group of data that is typically used to hold data that is related in some way.
